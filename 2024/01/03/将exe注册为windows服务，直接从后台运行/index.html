将exe注册为windows服务，直接从后台运行

使用 windows 自带的命令 sc

首先我们要打开 cmd，下面的命令在 cmd 中运行，最好使用管理员运行 cmd

注册服务：

参数 说明
type= own own - 指定在其自己的进程中运行的服务。 它不会与其他服务共享可执行文件。 这是默认值
start= auto auto-服务自动启动
binpath 你的应用程序所在的路径
displayname 服务显示的名称

sc.exe 配置 | Microsoft Learn

如何判断服务是否注册成功：

在 cmd 中输入services.msc打开系统服务，查看是否出现ceshi名称的服务（即displayname=后面的参数，我这里是ceshi）
